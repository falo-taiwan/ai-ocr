Gemini 3.5 OCR Lab

A public, neutral guide for calling Gemini vision models, optimizing A4 full-page images, and producing reviewable JSON drafts without exposing private workflows.

Main Route

gemini-3.1-flash-lite for low-cost first-pass OCR-style extraction.

Review Route

gemini-3.5-flash for stronger review when text is dense or uncertain.

Image Strategy

Use browser Canvas to create WebP images, with 768 px as the low-cost trial preset.

A4 Neutral Example

The public examples use generic A4 pages: workshop handouts, registration forms, warranty cards, and sign-in sheets.

Privacy Boundary

No hard-coded API keys. No private business documents. AI output stays draft until human review.

Recommended Request Pattern

A4 image
-> browser-side WebP optimization
-> gemini-3.1-flash-lite draft
-> optional gemini-3.5-flash review
-> human check
-> final JSON
