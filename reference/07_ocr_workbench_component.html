FALO OCR Workbench Component

OCR 是獨立資料轉換元件：圖片加 OCR 操作 Prompt，產生可編修文字結果。Prompt Manager 只負責呼叫它。

一句話

OCR Workbench 是獨立 PWA 元件，不是 Prompt Manager 主頁裡的欄位。主頁只保留入口按鈕與說明文字。

產品邊界

index.html

Prompt Manager 主頁只保留 OCR 入口與簡短說明。

Prompt Asset 變數 標籤 Workflow

ocr.html

OCR Workbench 管理圖片、OCR 操作 Prompt、輸出文字、模板匯入匯出。

Image + Prompt Editable Result CSV Template PWA

OCR Prompt 的意義

這裡的 OCR Prompt 是跟圖片一起送給 OCR 模型的操作指令，不是辨識後的文字整理 Prompt。

image / screenshot / handwritten note
        +
OCR operation prompt
        ↓
Gemini OCR or external OCR tool
        ↓
editable text result
        ↓
copy back to Prompt Manager

五組預設 Prompt

  • 一般文件 OCR
  • 表格 / 表單 OCR
  • 手寫筆記 OCR
  • 歷史文件 OCR
  • 單據 / 證據 OCR

Excel 模板閉環

OCR Prompt 模板使用 CSV 欄位，讓使用者可以匯出、用 Excel 或 Google Sheets 修改，再匯入全覆蓋。

id,title,description,prompt,outputFormat,useCase,status

這版不做 merge。全覆蓋比較容易教，也比較容易確認結果。

v0.3 範圍
