Platform Manifesto

FALO AI Capability Runtime 定位與世界觀

「OCR 是案例，Workflow 是主體，Governance 是靈魂，Knowledge Integration 才是長期護城河。」

專案定位：AI 能力服務治理平台

我們絕不能將此專案矮化為「OCR 辨識工具」。OCR 只是 Capability Card #1 (第一張能力卡)，是驗證架構、收取 API 差價與計算投資報酬率 (ROI) 的首個 PoC。

FALO 的核心定位是 Workflow × Governance × KM × ETL × AI Runtime。 我們在企業的 AI 應用流中，扮演「治理者」與「稽核者」的角色，將企業的知識、流程、文件、規則、AI 能力與審計要求無縫串接。

💡 客戶計費 vs. 夥伴對帳

客戶端採用直覺的 By-Case (按件/按頁) 定價，免去複雜算力解讀；而精細的內外雙軌對帳則是專為 FALO/TAAT 夥伴設計的後台治理利器，用以管控 API 成本與利潤空間。

📊 BI 數據分析與教材定位

此雙軌對帳與分流機制，不僅僅是系統後台，更是企業在介紹 BI 數據分析與 FinOps 營運決策時極佳的實戰教學案例，完美展示如何用數據驅動成本優化。

⚙️

核心方程式：Workflow × Governance × KM

Workflow (工作流是主體)

辨識文字只是起點，如何將資料自動帶入發票審核、ERP 報銷、跨系統派工等核心流程，才是企業的真正痛點。

Governance (治理是靈魂)

透過「內外帳對帳」與 API 代理防火牆，為企業監控每一次 API 呼叫的成功率、延遲、異常超扣與 Timeout，提供 FinOps 的最優調度。

KM (知識整合是長期護城河)

把所有經由能力卡過濾、ETL 結構化後的資料，沉澱為企業專屬的知識庫（如 NotebookLM 整合），形成越用越聰明的企業腦。

ETL & AI Runtime Audit

將非結構化文件（影像、聲音、合約）清洗為結構化資料，並在 Runtime 執行層記錄 Audit Trail，確保合規性與稽核軌跡。

Value Chain Lifecycle

FALO AI 企業數據價值鏈生命週期

從文件上傳開始，歷經 OCR 提取、結構化清洗、工作流流轉、ERP 記帳，最終沉澱為企業專屬知識庫與稽核 insights 的完整閉環。

數據生命週期說明：

  • 入流層 (ETL)：將企業複雜單據文件（影像）透過模組化能力卡與 AI Proxy 做高精準辨識並清洗為結構化 JSON 資料。
  • 業務層 (Workflow & ERP)：結構化資料直接回填企業現有工作流與 ERP，實現自動化財務報銷、進銷存登載。
  • 儲存層 (KM)：將所有單據與辨識脈絡沉澱至企業知識庫，作為未來企業內部搜尋、訓練、法規遵循的長期資產。
  • 治理層 (Audit & FinOps)：雙軌帳本引擎即時查帳，監控延遲、Timeout、異常扣款，提供成本分析與流程改善的管理決策洞察。

Evolutionary Roadmap

AI Runtime 與數據治理演進路線圖

從最初的單機工具，演進至具備自動對帳與 FinOps 成本治理的完整 FALO AI 流程與知識生態平台。

Phase 1: PoC 驗證期

FALO OCR Workbench (工具屬性)

完成單機版 PWA 容器、基礎 OCR 模板配置與 CSV 閉環匯出入，確認使用者能方便操作圖片資料轉換。

  • 單機 `ocr.html` 與 PWA 安裝支援
  • 五組內建 OCR Operation Prompts
  • Excel 式 CSV 全覆蓋導入導出
Phase 2: 服務化與雙軌帳本 (當前里程碑)

AI Proxy Gate & 雙軌帳本

導入 API 金鑰驗證，建構雙軌 ledger 寫入機制，將客戶使用端（內帳）與雲端 Provider（外帳）解耦，支援 AI 進行對帳審核與失敗重送路由。

  • AI Proxy Gateway 搭建與 API Key 認證
  • 內外雙軌 Ledger 同步寫入引擎
  • AI 自動對帳模組 (偵測超時、超收、退款存檔)
Phase 3: 工作流與 ERP 整合期

流程調度與自動化入帳

完善數據價值鏈。將結構化資料自動回寫入 ERP 系統，建立自動化單據核銷 workflow，並將歷史辨識文字儲存進企業知識庫。

  • 結合 ERP/會計軟體的數據回寫接口
  • 企業知識管理 (KM) 基礎儲存引擎
  • 自動核銷與流程審核 Workflow 整合
Phase 4: 全能力 Runtime 生態系

FALO AI Runtime & KM Market

擴展至多樣化能力卡（Vision, 翻譯, 語音, LLM），開放第三方上架，建構 Workflow × Governance × KM 的企業 AI 生態大腦。

  • 標準能力卡 SDK 與 Runtime 規範釋出
  • 多種 Capability Card 上架 (Speech, Audit)
  • 第三方 Capability Market 分成與訂閱結算機制
